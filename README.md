# 🚦 Road Accident Analysis Dashboard

An interactive Excel dashboard for analyzing road accident data across multiple dimensions — severity, vehicle type, road conditions, lighting, and area type — to support data-driven road safety decisions.

---

## 📊 Dashboard Overview

| Metric | Value |
|---|---|
| **Total Casualties** | 453,051 |
| **Fatal Casualties** | 8,277 (1.8%) |
| **Serious Casualties** | 66,298 (14.6%) |
| **Slight Casualties** | 378,476 (83.5%) |
| **Casualties by Cars** | 355,553 (78.5%) |

---

## 📌 Features

### 🔢 Primary KPIs
- Total, Fatal, Serious, and Slight casualties displayed as **number cards with donut charts**
- Casualties specifically by **Car** type with percentage breakdown

### 🚗 Casualties by Vehicle Type

| Vehicle | Casualties |
|---|---|
| Cars | 355,553 |
| Trucks | 38,098 |
| Motorcycles | 37,723 |
| Buses | 16,172 |
| Tractors | 1,352 |
| Others | 4,153 |

### 📈 CY vs PY Monthly Trend
- Line chart comparing **2020 vs 2021** month-by-month casualty trends
- Helps identify seasonal patterns and year-over-year changes

### 🛣️ Casualties by Road Type

| Road Type | Casualties |
|---|---|
| Single Carriageway | 335.0k |
| Dual Carriageway | 73.4k |
| Roundabout | 28.6k |
| One Way Street | 8.1k |
| Slip Road | 5.0k |
| Unknown | 2.9k |

### 🌧️ Casualties by Road Surface
- **Treemap** visualization showing Dry, Wet, Snow/Ice conditions
- Dry roads account for the majority of casualties

### 🏙️ Urban vs Rural Breakdown
- Donut chart: **Rural 274.3k** vs **Urban 178.7k**

### 🌙 Light Condition Analysis
- Donut chart: **Daylight 326.7k** vs **Darkness 126.3k**

### 🎛️ Filter Panel
Interactive slicers to filter all visuals by:
- **Accident Date / Year** (2020, 2021, 2022)
- **Wind Conditions** (High Winds, No High Winds, Other)
- **Weather Conditions** (Fine, Fog, Raining, Snowing, Other)
